## How to Use

### Download Manually

```bash
npx degit graphcms/graphcms-examples/with-sveltekit-and-urql with-sveltekit-and-urql
```

Install & Run:

```bash
cd with-sveltekit-and-urql
npm install
npm run dev -- --open
```

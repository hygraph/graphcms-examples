This example demonstrates how to query from GraphCMS with `graphql-hooks` in Next.js.

## How to Use

### Download Manually

```bash
npx degit graphcms/graphcms-examples/with-graphql-hooks with-graphql-hooks
```

Install & Run:

```bash
cd with-graphql-hooks
npm install
npm run dev
# or
cd with-graphql-hooks
yarn
yarn dev
```

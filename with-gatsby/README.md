## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-gatsby with-gatsby
```

Install & Run:

```bash
cd with-gatsby
npm install
npm run dev
# or
cd with-gatsby
yarn
yarn dev
```

## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-nuxtjs-i18n with-nuxtjs-i18n
```

Install & Run:

```bash
cd with-nuxtjs-i18n
npm install
npm run dev
# or
cd with-nuxtjs-i18n
yarn
yarn dev
```

### Generate static project

```bash
npm run generate
# or
yarn generate
```
